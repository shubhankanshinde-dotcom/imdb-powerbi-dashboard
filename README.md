## IMDb Top 1000 Movies — Power BI Dashboard

### About this project

This project explores the IMDb Top 1000 Movies dataset using Power BI to uncover trends in ratings, genres, runtime, certifications, and director performance.
The goal was to design a single-page interactive dashboard that allows users to analyze movie success metrics and identify patterns influencing audience preference and box office performance.

The dashboard combines KPIs, charts, and dynamic filters to support both casual movie exploration and business-oriented insights for streaming platforms, production studios, and analysts.

---

### Tools Used

* Power BI
* DAX (Calculated Measures)
* Data Cleaning & Transformation
* Excel / CSV Dataset

---

### Dataset

The dataset contains 1000 IMDb titles with attributes including:

* Movie title and release year
* IMDb rating
* Runtime
* Genre and certification
* Gross revenue
* Director

Data cleaning included handling missing values, formatting gross revenue, and standardizing categorical fields.

---

### What I worked on

* Data preparation and transformation in Power BI
* Creation of KPIs (Average Rating, Runtime, Total Gross, Title Count)
* DAX calculations for aggregated metrics
* Design of interactive visuals and slicers
* Dashboard layout and storytelling

---

### Key Insights

* Average IMDb rating across top movies is **7.95**
* Movies with runtime between **120–130 minutes** tend to receive higher ratings
* **PG-13 and U certifications** dominate in both popularity and revenue
* Directors like **Spielberg, Nolan, and Russo** lead in gross revenue
* Drama and Drama-Romance genres appear most frequently
